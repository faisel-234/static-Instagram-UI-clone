Instagram UI Clone (HTML & CSS Only)

A static Instagram-inspired user interface built using pure HTML and CSS.

This project focuses entirely on layout structuring, styling, and responsive design — without using JavaScript.

📌 Project Overview

This project is a frontend UI clone of Instagram created using only:

HTML5 for structure

CSS3 for styling and layout

The purpose of this project is to practice:

Layout replication

Flexbox positioning

Responsive design

Clean UI structuring

Visual accuracy

This is a static design project and does not include dynamic functionality.

✨ Features

📱 Responsive layout

🖼 Instagram-style post feed

👤 Profile section

🧭 Navigation bar

🎨 Clean and modern UI

💻 Built with pure HTML & CSS

🛠 Technologies Used

HTML5 – Semantic structure

CSS3 – Styling, Flexbox, Media Queries

No JavaScript.
No frameworks.
No libraries.
